# Google Translate Widget Demo

A minimal static GitHub Pages demo showing three Google Translate widget layouts:

- Vertical/default
- Horizontal
- Simple

The widgets allow translation between English, Polish, Spanish, Ukrainian, and Arabic.

## View the demo

Open `index.html` in a browser, or enable GitHub Pages for the repository and publish from the repository root.

## Implementation

The demo uses plain HTML, CSS, and JavaScript with no frameworks, build tools, or dependencies. All three widgets share one Google Translate callback and one script load.
